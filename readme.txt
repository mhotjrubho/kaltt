תוכנת kalttoffline
====================

תיאור כללי
-----------
זוהי מערכת אוף-ליין מלאה לניהול קמפיין התחייבויות ואיסוף תרומות בישיבה, עם שני מסכים עיקריים:
1. דשבורד ניהול על מחשב המנהל.
2. מסך תצוגה גדול לצופים.

המערכת עובדת ללא אינטרנט לחלוטין, כל הנתונים נשמרים מקומית במסד נתונים מקומי.

מאפיינים מרכזיים
-----------------
- RTL מלא בכל הממשק, כולל דשבורד, טפסים, טבלאות, גרפים והודעות.
- שימוש בתאריך עברי בכל מקום רלוונטי למשתמש (דוחות, תצוגות ונתונים).
- מסך תצוגה גדול המציג קבוצות, התחייבויות, יעד כללי ויעדים קבוצתיים.
- דשבורד לניהול קבוצות, בחורים, בוגרים, תרומות, ימי איסוף, הגרלות והודעות.
- ייצוא נתונים לקובצי PDF, כולל לפי קבוצה.
- עבודה מלאה אוף-ליין, ללא שרת חיצוני וללא תלות באינטרנט.
- שמירת נתונים מקומית לפי שנים, עם אפשרות בחירת שנה נוכחית.

התקנה והרצה (פיתוח)
---------------------
1. התקן Node.js על המחשב.
2. בתיקיית הפרויקט הרץ:
   npm install
3. להפעלת התוכנה במצב פיתוח:
   npm start

מסד נתונים ולוגים
------------------
- מסד נתונים SQLite נשמר בתיקייה data.
- לוגים נשמרים בתיקייה logs.
- לוגואים, רקעים ותמונות נשמרים בתיקייה assets.
